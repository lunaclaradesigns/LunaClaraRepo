# How to add your jewelry photos

Drag your image files into THIS folder (`public/products/`) using the
exact filenames below. The website picks them up automatically — no code
changes needed. Until a file exists, an elegant "Image coming soon"
placeholder shows in its place (so the site never looks broken).

Recommended: square-ish or portrait photos, at least 800×1000px, .jpg or .png.

## Product photos (individual items)

| Save your photo as...                      | Used for                          |
|--------------------------------------------|-----------------------------------|
| `earrings-vintage-floral.jpg`              | Vintage Floral Pearl Earrings     |
| `earrings-pearl-starburst.jpg`             | Starburst Pearl Drop Earrings     |
| `necklace-emerald.jpg`                     | Emerald Halo Necklace             |
| `necklace-gold-rose.jpg`                   | Golden Rose Locket Necklace       |
| `necklace-shell-pearl.jpg`                 | Shell & Pearl Necklace            |
| `bangle-crystal-gold.jpg`                  | Crystal Wrap Bangle               |
| `ring-solitaire.jpg`                       | Solitaire Crystal Ring            |

## Logo

Save your logo in the `public/` folder (one level up from here):

| Save your logo as...     | Used for                                  |
|--------------------------|-------------------------------------------|
| `public/logo.png`        | Navbar (use a white or transparent bg)    |
| `public/logo-gold.png`   | Footer (transparent bg, gold text) — optional |

If you don't add a logo file, the built-in gold SVG logo is used automatically.
